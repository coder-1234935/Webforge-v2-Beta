# WebForge V2 (Beta)

Welcome to the official repository for **WebForge V2 Beta**, the next-generation web development and design platform powered by **Opentra Studio**. This version serves as the bleeding-edge release candidate for the standard WebForge platform.

## 🚀 Overview

WebForge V2 Beta is designed for users who want to experience the latest features, performance improvements, and tools before they hit the public version. This version is actively maintained and receives updates significantly earlier than the public release.

*   **Beta Live URL:** [forgeweb-v2-beta.lovable.app](https://forgeweb-v2-beta.lovable.app/)
*   **Public Version:** [forgeweb.lovable.app](https://forgeweb.lovable.app/)
*   **Public Repo:** [coder-1234935/webforge-public](https://github.com/coder-1234935/webforge-public)
*   **Organization:** [Opentra Studio](https://opentrastudio.netlify.app/)

## ✨ Key Features

*   **Priority Updates:** V2 Beta users receive the latest patches, features, and engine optimizations prior to the public GitHub version.
*   **Advanced Modules:** Access to experimental UI components and backend integrations currently in the testing phase.
*   **Enhanced Stability:** Performance-focused builds optimized for professional web development workflows.

## 🔑 Access Control

> [!IMPORTANT]
> **WebForge V2 Beta is a paid version.** 
> While the application is live, it requires a **License Key** for access. 

To obtain a key and start forging:
1. Visit the [Opentra Studio Official Site](https://opentrastudio.netlify.app/).
2. Request access through the Beta program dashboard.
3. Enter your key at the [V2 Login Portal](https://forgeweb-v2-beta.lovable.app/).

## 🛠 Tech Stack

WebForge is built with a focus on speed, scalability, and modern user experience:
*   **Framework:** React / Vite
*   **Styling:** Tailwind CSS
*   **Platform:** Lovable.dev
*   **Deployment:** Vercel / Netlify
